databaseGetChairs R Documentation

Returns list of chairs on a specified faculty

Description

Returns list of chairs on a specified faculty

Usage

databaseGetChairs(
  faculty_id = "",
  offset = "",
  count = "100",
  v = getAPIVersion()
)

Arguments

faculty_id

ID of the faculty to get chairs from
offset

Offset required to get a certain subset of chairs
count

Amount of chairs to get
v

Version of API
