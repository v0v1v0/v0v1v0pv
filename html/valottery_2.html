cash.5.1xday R Documentation

Cash 5 (once daily)

Description

Historical data for the Cash 5 once daily game. Game Play: Pick five numbers from 1 through 34. Note: On April 11, 1999, Cash 5 switched to twice daily drawings.

Usage

cash.5.1xday

Format

A data frame with 1187 rows and 6 variables:

date

date of draw

N1

1st number in order

N2

2nd number in order

N3

3rd number in order

N4

4th number in order

N5

5th number in order
