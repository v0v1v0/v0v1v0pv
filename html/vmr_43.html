vmrUpdatePackages R Documentation

Update R packages installed

Description

Updates R packages installed in the guest machine.

Usage

vmrUpdatePackages()

Details

Will perform a update.packages() in the guest machine of the current vmr environment.

Examples

## Not run: 
lboxes <- vmrList()
vmr <- vmrCreate(lboxes$Name[1])
vmr <- vmrInitEnv(vmr)
vmrStart()
vmrUpdatePackages()

## End(Not run)
