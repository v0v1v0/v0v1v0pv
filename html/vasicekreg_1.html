vasicekreg-package R Documentation

Overview of the vasicekreg package

Description

The vasicekreg package implements the probability density function, quantile function, cumulative distribution function and random number generation function for Vasicek distribution parameterized, either, as a function of its mean or its \tau-th quantile, 0 <\tau<1. In addition, two gamlss frameworks for regression analysis are available. Some function are written in C++ using Rcpp.

Details

bodyfat: Body fat data set.

VASIM: For mean modeling (con/in)ditional on covariate(s).

VASIQ: For quantile modeling (con/in)ditional on covariate(s).
