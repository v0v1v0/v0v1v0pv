vmrSetVerbose R Documentation

Set verbose level

Description

Set verbose level for vmr package functions

Usage

vmrSetVerbose(verbose_mode = "Normal")

Arguments

verbose_mode

"None", "Normal" or "Full"

Details

Three verboses mode is available:

  • "None" : print nothings

  • "Normal" : print essential

  • "Full" : print all

Value

invisible verbose value
