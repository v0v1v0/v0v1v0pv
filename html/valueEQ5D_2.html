Description

Function to check the given column exists

Usage

check_column_exist(column_name, data)

Arguments

column_name

a column name
data

data frame

Value

0 if success -1 if failure

Examples

check_column_exist("age", data.frame(
  age = rep(20, 4), sex = rep("male", 4),
  stringsAsFactors = FALSE
))
