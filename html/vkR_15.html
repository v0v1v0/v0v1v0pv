databaseGetRegions R Documentation

Returns a list of regions

Description

Returns a list of regions

Usage

databaseGetRegions(
  country_id = "",
  q = "",
  offset = "",
  count = "100",
  v = getAPIVersion()
)

Arguments

country_id

Country ID, received in database.getCountries method
q

Search query
offset

Offset needed to return specific subset of regions
count

Number of regions to return
v

Version of API
