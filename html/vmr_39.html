vmrStop R Documentation

Stop a vmr environement

Description

Stop a guest virtual machine in the current vmr environment.

Usage

vmrStop(force = FALSE)

Arguments

force

if TRUE force to stop (powerOff), otherwise FALSE clean shutdown

Value

NULL

Examples

## Not run: 
lboxes <- vmrList()
vmr <- vmrCreate(lboxes$Name[1])
vmr <- vmrInitEnv(vmr)
vmrStart()
vmrStop()

## End(Not run)
