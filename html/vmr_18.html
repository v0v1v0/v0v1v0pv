vmrIsRunning R Documentation

Is vmr environment running

Description

Check if a guest machine in a vmr environment is running

Usage

vmrIsRunning()

Value

TRUE if running, otherwise FALSE

Examples

## Not run: 
lboxes <- vmrList()
vmr <- vmrCreate(lboxes$Name[1])
vmr <- vmrInitEnv(vmr)
vmrStart()
vmrIsRunning()
vmrStop()
vmrIsRunning()

## End(Not run)
