get_samvatsara_name R Documentation

get_samvatsara_name

Description

Name of the Shaka Samvatsar for a given Julian day number and maasa number.

Usage

get_samvatsara_name(jd, maasa_num)

Arguments

jd

Julian day number
maasa_num

Maasa number

Value

Shaka Samvatsar

Examples

get_samvatsara_name(2459778,2)
