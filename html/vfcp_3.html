kopula R Documentation

Copula object

Description

Generate the copula object.

Usage

kopula(fam, tht, dm)

Arguments

fam

Family name copula. These can be: "clayton", "gumbel", "frank", "joe", "amh", "fgm".

tht

Copula parameter.

dm

Copula dimension.

Value

Copula object
