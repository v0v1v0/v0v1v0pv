temp_veh R Documentation

Expanded Vehicles data.frame by hour

Description

temp_veh multiplies vehicles with temporal factor

Usage

temp_veh(x, tfs, array = FALSE)

Arguments

x

Vehicles data.frame
tfs

temporal factor
array

Logical, to return an array

Value

data.table

See Also

temp_fact

Examples

## Not run: 
data(net)
data(pc_profile)
x <- age_ldv(x = net$ldv)
dx <- temp_veh(x = x, tfs = pc_profile[[1]])
plot(Vehicles(as.data.frame(dx[, 1:50])))
dx2 <- temp_veh(x = x,
                tfs = pc_profile[[1]],
                array = TRUE)
plot(EmissionsArray(dx2))

## End(Not run)
