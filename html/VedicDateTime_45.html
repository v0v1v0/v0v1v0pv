yoga R Documentation

yoga

Description

Yoga for a given place and time

Usage

yoga(jd, place)

Arguments

jd

Julian day number
place

Vector containing latitude, longitude and timezone

Value

Yoga and it's ending time

Examples

yoga(2459778,c(15.34, 75.13, +5.5))
yoga(gregorian_to_jd(17,6,2022),c(15.34, 75.13, +5.5))
