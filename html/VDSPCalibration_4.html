samplefun R Documentation

Uniformly Sampling

Description

Draws samples uniformly (for internal use only)

Usage

samplefun(x, index, n0)

Arguments

x

The VD values
index

the index for VD value, it can be 1, 2, 3,....
n0

Sample size

Details

Uniform sampling (internal use only)

Value

index

selected ids

x

selected VD levels

Author(s)

Durazo-Arvizu, Ramon, Sempos, Chris and Tian, Lu

See Also

sampletot

Examples

x=rnorm(100)
index=1:100
samplefun(x, index, 40)
