SurvData R Documentation

Simulated Survival Data

Description

This dataset contains survival outcome.

Usage

data(VBJMdata)

Format

A data frame with 100 rows and 4 variables

Details

  • ID patient ID

  • fstat censoring indicator

  • ftime survival time

  • x baseline covariates
