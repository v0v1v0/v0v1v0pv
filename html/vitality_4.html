ft.4p R Documentation

Density function for 3-parameter (r, s, u)

Description

This function is used in the calculation of the fitted intrinsic (mu.vd1.4p) and total (mu.vd.4p) mortality rate in the 4-parameter model.

Usage

  ft.4p(xx, r, s, u)

Arguments

xx

age
r

r value
s

s value
u

u value

Value

density
