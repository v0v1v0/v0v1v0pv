ft.6p R Documentation

Density function for 2-parameters (r, s)

Description

This function is used in the calculation of the fitted intrinsic (mu.vd1.6p) and total (mu.vd.6p) mortality rate in the 6-parameter model.

Usage

  ft.6p(xx, r, s)

Arguments

xx

age
r

r value
s

s value

Value

density
