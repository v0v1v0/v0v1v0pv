write_list R Documentation

Writes a list to a path

Description

Writes a list to a path

Usage

write_list(x, path)

Arguments

x

A list.
path

A full path to file.

Value

A file named 'list.txt' in 'path'.

Examples

## Not run: 
library(voice)

pts <- list(x = cars[,1], y = cars[,2])
listFile <- paste0(tempdir(), '/list.txt')
voice::write_list(pts, listFile)
file.info(listFile)
system(paste0('head ', listFile))
