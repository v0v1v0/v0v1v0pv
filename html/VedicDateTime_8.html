get_nakshatra_name R Documentation

get_nakshatra_name

Description

Get name(s) of the Nakshatra for given Julian day number and place.

Usage

get_nakshatra_name(jd, place)

Arguments

jd

Julian day number
place

Vector containing latitude, longitude and timezone

Value

Name(s) of the Nakshatra and its ending time.

Examples

get_nakshatra_name(2459778,c(15.34, 75.13, +5.5))
get_nakshatra_name(swephR::swe_julday(2022,7,14,0,swephR::SE$GREG_CAL),c(15.34, 75.13, +5.5))
