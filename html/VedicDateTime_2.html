day_duration R Documentation

day_duration

Description

Duration of the day for a given place and time

Usage

day_duration(jd, place)

Arguments

jd

Julian day number
place

Vector containing latitude, longitude and timezone

Value

Vector containing the length of the day & in dms

Examples

day_duration(2459778,c(15.34, 75.13, +5.5))
day_duration(swephR::swe_julday(2022,7,14,0,swephR::SE$GREG_CAL),c(15.34, 75.13, +5.5))
