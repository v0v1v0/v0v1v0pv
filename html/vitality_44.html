vft.6p R Documentation

Vectorized density function

Description

This function is used in the calculation of the fitted intrinsic (mu.vd1.6p) and total (mu.vd.6p) mortality rate in the 6-parameter model.

Usage

  vft.6p(xx, r, s)

Arguments

xx

vector of ages
r

r value
s

s value

Value

vector of densities
