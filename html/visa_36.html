SpectraMatrix-class R Documentation

Class 'SpectraMatrix'

Description

SpectraMatrix is a extended 'Spectra' class.

Constructor as.spectra.matrix creates a SpectraMatrix object.

Usage

as.spectra.matrix(
  spectra = matrix(0),
  wavelength = numeric(0),
  w.unit = character(0)
)

Arguments

spectra

A matrix
wavelength

A numeric vector
w.unit

A character string

Value

sdf

Returns a SpectraDataFrame.

Examples

smatrix <- as.spectra.matrix(matrix(1:10, 1), 1:10, "nm")
str(smatrix)
