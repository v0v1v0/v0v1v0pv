nakshatra R Documentation

nakshatra

Description

Nakshatra for a given place and time

Usage

nakshatra(jd, place)

Arguments

jd

Julian day number
place

Vector containing latitude, longitude and timezone

Value

Nakshatra and it's ending time

Examples

nakshatra(2459778,c(15.34, 75.13, +5.5))
nakshatra(gregorian_to_jd(17,6,2022),c(15.34, 75.13, +5.5))
