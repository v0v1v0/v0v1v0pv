d_to_nnt R Documentation

Convert d to Number Needed to Treat

Description

This function calculates the number needed to treat from continuous measures (Cohen's d) using Kraemer and Kupfer (2006) formula.

Usage

d_to_nnt(d = NULL)

Arguments

d

the effect size

Value

nnt values from d
