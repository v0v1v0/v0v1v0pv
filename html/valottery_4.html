decades.of.dollars R Documentation

Decades of Dollars

Description

Historical data for the Decades of Dollars game. You pick six (6) numbers, the Lottery will then select six (6) numbered balls. Note: This game was discontinued in favor of CASH4LIFE.

Usage

decades.of.dollars

Format

A data frame with 443 rows and 7 variables:

date

date of draw

N1

1st number in order

N2

2nd number in order

N3

3rd number in order

N4

4th number in order

N5

5th number in order

N6

6th number in order

Source

https://www.valottery.com
