usersGetFollowers R Documentation

Returns a list of IDs of followers of the user in question, sorted by date added, most recent first

Description

Returns a list of IDs of followers of the user in question, sorted by date added, most recent first

Usage

usersGetFollowers(
  user_id = "",
  offset = 0,
  count = 0,
  fields = "",
  name_case = "",
  drop = FALSE,
  flatten = FALSE,
  progress_bar = FALSE,
  v = getAPIVersion()
)

Arguments

user_id

User ID
offset

Offset needed to return a specific subset of followers
count

Number of followers to return
fields

Profile fields to return
name_case

Case for declension of user name and surname
drop

Drop deleted or banned followers
flatten

Automatically flatten nested data frames into a single non-nested data frame
progress_bar

Display progress bar
v

Version of API

Examples

## Not run: 
my_followers <- usersGetFollowers(me())

## End(Not run)
