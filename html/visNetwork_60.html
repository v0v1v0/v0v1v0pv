Description

Function to remove edges from network, with shiny only.

Usage

visRemoveEdges(graph, id, legend = FALSE)

Arguments

graph

: a visNetworkProxy object
id

: vector of id, edges to remove
legend

: Boolean. Remove edges on legend ? Default to FALSE
