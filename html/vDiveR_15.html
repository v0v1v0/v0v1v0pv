plot_worldmap R Documentation

Geographical Distribution of Sequences Plot

Description

This function plots a worldmap and color the affected geographical region(s) from light (lower) to dark (higher), depends on the cumulative number of sequences. Aside from the plot, this function also returns a dataframe with 2 columns: 'Region' and 'Number of Sequences'. The input dataframe of this function is obtainable from metadata_extraction(), with NCBI Protein / GISAID EpiCoV FASTA file as input.

Usage

plot_worldmap(meta, base_size = 8)

Arguments

meta

a dataframe with 3 columns, 'ID', 'region', and 'date'
base_size

word size in plot

Value

A list with 2 elements (a plot followed by a dataframe)

Examples

geographical_plot <- plot_worldmap(metadata)$plot
geographical_df <- plot_worldmap(metadata)$df
