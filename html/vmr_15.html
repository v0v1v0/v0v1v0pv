vmrInfo R Documentation

Get guest machine information

Description

Get guest machine information. Print OS, R, R-devel and R packages information. Still in development.

Usage

vmrInfo()

Value

NULL

Examples

## Not run: 
boxes <- vmrList()
vmr <- vmrCreate(boxes$Name[1])
vmr <- vmrInitEnv(vmr)
vmrStart()
vmrInfo()

## End(Not run)
