.get_EQ5D_value_sets R Documentation

.get_EQ5D_value_sets

Description

This function generates all EQ states based on the provided EQ5D version and adds utilities.

Usage

.get_EQ5D_value_sets(version, value_sets, value_sets_XW)

Arguments

version

A character string specifying the EQ5D version.
value_sets

A character vector specifying the country value sets for the given EQ5D version.
value_sets_XW

A character vector specifying the country value sets for the crosswalk EQ5D version.

Value

A list containing:

  • utilityColumn: A character vector with the names of the added utility columns.

  • df: A data frame containing all generated EQ states with their utilities.
